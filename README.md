# A countdown bot for Discord

Firstly, let's countdown to Inauguration 2021. 
In the future, let users submit countdown counters of their own.

This simple bot is heavily based on Coding Train discord bot.

This URL is used to add the bot to discord servers.
https://discord.com/oauth2/authorize?client_id=CLIENT_ID&scope=bot

# dates
A directory with YYYYMMDD.txt files with events.

# dates-api
A Rust language WASM program that provides a RESTful interface for date management

# discord-countdown-rs
A Rustlang discord bot that can interface with the dates and give various countdowns.  It can also display the contents of the media.txt file.
